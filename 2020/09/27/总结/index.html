面向对象首先创建一个新的 class类。创建的类没有主入口方法。这个class类就是一个对象，它有三个属性 名字年龄长度

package helloword;

public class entity {

String name;

int age;

int length;

public entity() {

// TODO Auto-generated constructor stub

}

public String getName() {

return name;

}

public void setName(String name) {

this.name = name;

}

public int getAge() {

return age;

}

public void setAge(int age) {

this.age = age;

}

public int getLength() {

return length;

}

public void setLength(int length) {

this.length = length;

}

}

构造器发方法 特点：没有返回值，构造器的名字跟类名一致。

package helloword;

public class entity2 {

String name;

int age;

int length;

public entity2() {

// TODO Auto-generated constructor stub

}

public entity2(String name) {

super();

this.name = name;

}

public entity2(String name, int age) {

super();

this.name = name;

this.age = age;

}

public entity2(String name, int age, int length) {

super();

this.name = name;

this.age = age;

this.length = length;

}

@Override

public String toString() {

return “entity2 [name=” + name + “, age=” + age + “, length=” + length + “]”;

}

}

entity2 xiaomi = new entity2(“小米”,15,55); 在创建entity2的同时给这个对象的属性赋值
